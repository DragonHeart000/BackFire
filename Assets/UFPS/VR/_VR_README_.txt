--- REQUIREMENTS ---

* The UFPS VR integration requires at least Unity 5.3.4f1.

* These scripts were developed on the Oculus platform using "Oculus Utilities for Unity 5 V1.5.0"

* You can find the "Oculus Utilities for Unity" here:
	https://developer.oculus.com/downloads/

* You can find the current Unity version requirements of Oculus here:
	https://developer.oculus.com/documentation/game-engines/latest/concepts/unity-integration-req/

--- SETUP ---

* Please refer to the UFPS VR manual chapter for info on how to set up UFPS with Oculus.
	http://www.opsive.com/assets/UFPS/hub/assets/ufps/vr/

* Before playing the demo scenes, please see the detailed setup instructions for each in the 'Scenes' folder.
